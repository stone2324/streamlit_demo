# streamlit_demo
Streamlit Demo

This Streamlit application displays two interesting lists:
1. **Exotic Fruits in Asia** - A list of exotic Asian fruits my friend should try
2. **AI Techniques & Use Cases** - Popular AI techniques and their practical applications

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit app:

```bash
streamlit run streamlit_app.py
```

The app will open in your default web browser at `http://localhost:8501`
